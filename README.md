# SearchHub - Simplifying PDF Interactions

## Setup and Configuration

### Installation

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/your-username/searchhub.git

2. Install the necessary Python dependencies using pip and the requirements.txt file
      
    pip install -r requirements.txt

3. Running the Application
  Start the SearchHub web application by running the following command

     **streamlit run main.py

   Once the application is running, open a new terminal window.

Navigate to the okta-flask-quickstart folder.

Create a Python virtual environment (venv) for the Okta Flask application
    
     1.cd okta-flask-quickstart
     2.python -m venv venv

4.  Activate the virtual environment. Use the appropriate command based on your operating system

    For Linux or macOS- . venv/bin/activate

    For Windows-  venv\Scripts\activate.bat or venv\Scripts\activate

5. Install additional dependencies required for the Okta Flask application
 
      pip install requests Flask flask-cors pyOpenSSL Flask-Login

6. Start the Okta Flask application by running

      flask run
The Okta Flask application will run and provide a hosting link. Open a web browser and go to this link (http://localhost:5000).

 you're ready to use SearchHub to interact with your PDF documents
